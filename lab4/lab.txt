1. Как увеличить размер (шрифт) маркеров?
2. Как задать начальное значение для нумерации списка?
3. Как задать цвет текста для списка?
4. Какова структура не нумерованного списка?
5. Как изменить внешний вид маркера?
6. Какова структура вложенного списка?
7. Как расположить два списка рядом друг с другом, один список в левой части страницы, другой в правой?
8. Как создать стиль для нумерованного списка?

ANSWERS:
1. Изменить размер (шрифт) маркеров можно с помощью аттрибутов или же стилей
2. Задать начальное значение можно с помощью аттрибута start
3. Цвет можно задать с помощью аттрибутов или же стилей
4. Сначала объявляется тип списка, в данном случае тегом <ul>, затем указываем каждый пункт с помощью тега <li>Пункт</li>. А в конце тегом </ul> закрываем наш не нумерованный список.
5. Внешний вид можно поменять с помощью аттрибута type, а также стилей.
6. Сначала объявляется тип списка, в данном случае тегом <ol>, затем указываем каждый пункт с помощью тега <li>Пункт</li>. А в конце тегом </ol> закрываем наш нумерованный список.
7. Чтобы расположить два списка рядом можно использовать таблицы и/или css.
8. Чтобы создать стиль для списка нужно указать класс либо идентификатор
